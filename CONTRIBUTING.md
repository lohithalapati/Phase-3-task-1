# Contributing to NeuralHandoff V5

Thank you for your interest in contributing!

## Workflow
1. Fork the repository.
2. Create a new branch: \git checkout -b feature/your-feature-name\ 
3. Make your changes.
4. Create a Pull Request against the \main\ branch.

## Standards
- Adhere to the rules in \docs/CODING_STANDARDS.md\.
- If making a significant architectural change, create an ADR in \docs/adr\.
