# Definition of Done

Every task, story, or bug fix must satisfy the following criteria before being considered complete:

- ✅ **Build Passes:** \
pm run build\ completes without errors.
- ✅ **TypeScript Clean:** \
px tsc --noEmit\ reports 0 errors.
- ✅ **Lint Clean:** \
pm run lint\ reports 0 issues.
- ✅ **Documentation Updated:** Relevant ADRs, READMEs, or diagrams are updated.
- ✅ **Git Committed:** Work is committed with a conventional commit message.
- ✅ **PR Approved:** Pull Request is self-reviewed and approved by at least one other team member.
