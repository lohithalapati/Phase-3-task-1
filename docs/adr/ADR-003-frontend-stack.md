# ADR-003: Core Frontend Stack

**Status:** Accepted

## Context
Defining the core technology stack is critical for consistency and development velocity.

## Decision
Our stack is locked as: **Vite** (build tool), **React** (UI library), **TypeScript** (type safety), **Tailwind CSS** (utility-first styling), **Zustand** (state), **React Hook Form** (forms), and **Zod** (validation).

## Consequences
This stack is optimized for developer experience, performance, and type safety. All new development must conform to this stack.
